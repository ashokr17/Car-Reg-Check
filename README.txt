Pre-Requisites:
---------------

1. Python 3
2. Pycharm
3. Necessary packages to be installed


User Configuration:
-------------------

User configures the following variables

1. inputPath - directory with multiple input files(car_input_v2.txt, car_input_v1.txt, etc)
2. outputPath - absolute path of output file(car_output_v2.txt)
3. Mention the path for chromedriver.exe in the file.

Results:
--------

1. The script creates a html report in the same folder, test_results.html with the details of Pass/Fail for each test based on the comparison with the output file.
2. Total count of all tests and pass/fail tests will be displayed in the console.
